Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan menstrukturkan halaman web. Sejarah HTML dimulai pada awal 1990-an, ketika seorang fisikawan bernama Tim Berners-Lee menciptakan protokol World Wide Web (WWW) di CERN (Organisasi Eropa untuk Riset Nuklir).

Berikut ini adalah sejarah perkembangan HTML

1. HTML 1.0 (1991-1992):

Versi pertama HTML dirilis oleh Tim Berners-Lee pada tahun 1991. Versi ini sangat sederhana, memungkinkan para pengguna membuat dokumen teks yang berisi tautan (hyperlink) untuk menghubungkan dokumen lain di internet.

2. HTML 2.0 (1995):

Pada tahun 1995, HTML 2.0 diperkenalkan sebagai standar oleh Internet Engineering Task Force (IETF). Versi ini memperbaiki dan menstandarkan elemen-elemen HTML yang sebelumnya telah digunakan secara tidak resmi. HTML 2.0 mendukung form dan elemen dasar lainnya.

3. HTML 3.2 (1997):

World Wide Web Consortium (W3C), yang dibentuk oleh Berners-Lee, mengambil alih pengembangan standar HTML. HTML 3.2 membawa banyak peningkatan, termasuk dukungan untuk tabel, scripting (melalui JavaScript), dan gaya (CSS).

4. HTML 4.0 (1997):

Dirilis pada akhir 1997, HTML 4.0 menambahkan banyak fitur baru, termasuk lebih banyak dukungan untuk aksesibilitas, form, CSS untuk pemisahan konten dan tampilan, serta integrasi yang lebih baik dengan JavaScript.

5. XHTML (2000):

XHTML (Extensible HyperText Markup Language) adalah upaya untuk lebih merapikan kode HTML agar lebih sesuai dengan standar XML. XHTML lebih ketat dalam hal penulisan dan struktur kode, serta lebih baik dalam memastikan kompatibilitas antara berbagai perangkat dan browser.

6. HTML5 (2014):

HTML5 merupakan evolusi signifikan dari HTML, dirancang untuk menggantikan HTML4 dan XHTML. Dirilis secara resmi pada tahun 2014 oleh W3C, HTML5 membawa banyak fitur baru seperti dukungan video dan audio bawaan, grafis 2D dan 3D (melalui elemen canvas), serta peningkatan dalam interaktivitas dan penyimpanan offline. HTML5 juga dirancang untuk mendukung aplikasi web yang lebih kaya dan lebih canggih tanpa memerlukan plugin tambahan seperti Flash.
